Ranqueamento de potenciais clientes

A empresa de seguro de saúde Insurance All está considerando a possibilidade de expandir seus serviços para incluir um novo produto: seguro de automóveis. O objetivo é fornecer uma lista de potenciais clientes, dentro do limite de 20.000 pessoas, para a realização de uma campanha telefônica.


As ferramentas utilizadas foram:

  • Python, Pandas, Numpy, Matplotlib e Seaborn
  • Pacotes de Machine Learning: Sklearn
  • Algoritmos de Classificação: KNN Classifier, Logistic Regression, XGBoost Classifier, Random Forest Classifier, Extra Tree Classifier, Decision Tree Classifier
  • Python API's
  • Git e Github
  • Jupyter Notebook e VS Code

Na resolução deste projeto foi feita a descrição dos dados, criação de hipóteses, preparação dos dados, seleção de features, teste com diferentes algoritmos de machine learning e ajustes dos hiperparâmetros. Ao final foram respondidas as perguntas de negócio e colocado em produção, podendo acessar as predições através do Google Sheets (vídeo exemplo).


Resultado: Com 20.000 ligações o time de vendas conseguirá contatar 86,3% do total de interessados. Sendo mais de 2x melhor do que se fossem feitas ligações aleatórias.

Identificação de imóveis para compra e revenda a fim de maximizar o lucro

O estudo envolveu a resposta de duas perguntas importantes:

  • Quais são os imóveis que deveriam ser comprados e por qual preço?
  • Uma vez que o imóvel seja comprado, qual o melhor momento para vendê-lo e por qual preço?

Para responder essas perguntas, foram adotadas algumas premissas e realizado um processo com 7 passos.


As ferramentas utilizadas foram:

  • Python 3.10.1
  • Pandas, Numpy e Seaborn
  • Mapas interativos com Plotly e Folium
  • Streamlit
  • Streamlit Cloud
  • Google Colab e VS Code

O resultado final do projeto incluiu dois relatórios, um com sugestões de compra de apartamentos com valores recomendados e outro com sugestões de venda de imóveis por valores recomendados, além de um dashboard.

As principais descobertas incluem insights sobre o valor médio da área quadrada de imóveis com vista para o mar, a variação do valor de imóveis dependendo da localização e a diferença no valor médio da área quadrada de imóveis em condições muito boas e em condições médias.


Resultado: Foram listados 743 imóveis como passíveis de serem comprados. Estimasse um lucro de $ 102.839.213,40 com a revenda deles.
